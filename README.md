# 💧 Water Quality Prediction & Classification
**A Machine Learning Project for Industrial Water Safety**

## 📌 Project Overview
This project focuses on predicting **Water Potability** (safe for human consumption) using physical and chemical water quality metrics. As an aspiring Data Scientist, I built this model to demonstrate how machine learning can be applied to environmental technology—a core focus area for global leaders like **XYLEM**.

## 🚀 Technical Stack
*   **Language:** Python 3.x
*   **Platform:** Google Colab
*   **Libraries:** 
    *   `Pandas` (Data Manipulation & Cleaning)
    *   `Scikit-Learn` (Machine Learning & Evaluation)
    *   `NumPy` (Numerical Processing)
    *   `Matplotlib` (Data Visualization)

## 📊 Methodology
1.  **Data Cleaning:** Handled missing values for critical features like pH and Sulfate using median imputation to prevent data bias.
2.  **Exploratory Data Analysis (EDA):** Analyzed the distribution of chemical properties across 3,000+ water samples.
3.  **Model Selection:** Implemented a **Random Forest Classifier** to handle the non-linear relationships between water parameters.
4.  **Evaluation:** Achieved a baseline accuracy of **66.62%**, focusing on model reliability and predictive consistency.

## 📈 Key Insights
*   The model identified **Sulfate levels** and **pH** as the most significant predictors of whether water is safe to drink.
*   Ensemble methods (Random Forest) performed better than simple linear models due to the complexity of environmental data.

## 📂 How to Run
1. Clone this repository.
2. Upload the `water_potability.csv` dataset.
3. Open the `.ipynb` notebook in Google Colab or Jupyter and run all cells.
